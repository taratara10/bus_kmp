# app-wearos 画面仕様書

## 概要

最寄りのバス停の時刻表を表示する Wear OS アプリ。
滝野井・津田沼の 2 バス停について、次に乗れるバスの発車時刻をリアルタイムで確認できる。

---

## 画面一覧

### 1. メイン画面

アプリ起動時に表示される画面。2 つのバス停を左右スワイプで切り替えられる。

#### レイアウト

```
┌──────────────────────────┐
│                          │
│  ← スワイプで切替 →      │
│  [バス停ページ]           │
│                          │
│        [●] [○]           │  ← 現在ページのインジケーター
└──────────────────────────┘
```

#### 機能

- 左右スワイプで「滝野井」と「津田沼」のページを切り替える
- 画面下部のドットで現在どちらのページを見ているか分かる
- アプリがフォアグラウンドにある間、1 秒ごとに表示を自動更新する
- アプリを前面に戻したとき（画面を上げたとき）に最新データを再取得する
- バックグラウンドに移ったときは自動更新を停止する

---

### 2. バス一覧ページ（各バス停）

スワイプで表示されるバス停ごとのページ。最大 5 件の次のバスが一覧表示される。

#### レイアウト

```
┌──────────────────────────┐
│  滝野井                  │
│  ─────────────────────  │
│  ┌────────────────────┐  │
│  │ ●  10:30    2分後  │  │  ← 次のバス
│  │      02:45         │  │
│  └────────────────────┘  │
│  │                       │
│  ◯  10:45    15分後      │  ← 次以降のバス
│  │                       │
│  ◯  11:00    30分後      │
│  ...                     │
└──────────────────────────┘
```

#### 機能

- バス停名を先頭に表示する
- 現在時刻より後に出発するバスのみを最大 5 件表示する
- バスが 1 本もない場合は「No Bus」と表示する
- スマートウォッチのクラウン（リューズ）やベゼルを回してスクロールできる
- バス同士は縦線でつながれ、次のバスは塗り潰し円（●）、以降はリング（◯）で区別される

---

### 3. 次のバス表示（ハイライトカード）

一覧の先頭に表示される、最も近い発車便。

```
┌─────────────────────────┐
│ ●  10:30        路線 07 │
│       02:45             │
└─────────────────────────┘
```

| 要素 | 内容 |
|---|---|
| 発車時刻 | `HH:mm` 形式 |
| カウントダウン | 発車まで何分何秒か（`MM:ss` 形式、1 秒ごとに更新） |
| 路線名 | 路線の番号・名称 |

ボーダーと大きいフォントで他のバスと視覚的に区別される。

---

### 4. 次以降のバス表示（通常カード）

2 件目以降のバス。

```
◯  10:45       15分後
   路線 08
```

| 要素 | 内容 |
|---|---|
| 発車時刻 | `HH:mm` 形式 |
| 残り時間 | 発車まで何分か（`X分後` テキスト） |
| 路線名 | 路線の番号・名称 |

---

## バス停・路線

| バス停 | 対応路線 |
|---|---|
| 滝野井 | 07・08 |
| 津田沼 | 05/06・07・08 |

平日・休日（土日）で異なる時刻表を自動で切り替える。切り替えはシステムの日付に基づく。
